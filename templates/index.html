LSB Steganogrpahy

LSB stands for Least Significant Bit. It is a way to embed secret information by replacing the minimm weighting value of a sampled speech signal with binary bits of secret information data, the secret information can be hidden in the speech.

Image Steganogrpahy

As the name suggests, Image steganography refers to the process of hiding data with an image file. The image selected for this purpose is called the cover image and the image obtained after steganography is called the stego image.
An image is represented as an N*M (in case of greyscale images) or N*M*3 (in case of color images) matrix in memory, with each entry representing the intensity value of a pixel. In lsb image steganography, a message is embedded into an image by altering the values of the lsb pixels, which are chosen by an encryption algorithm. The recipient of the image must be aware of the same algorithm in order to known which pixels he or she must select to extract the message.

Image Embedding

As the name suggests, a secret image is embedded onto a cover image of your liking. This is done by embedding the two most significant bits of the secret image in the two least significant bits of the cover image. Your watermarked image will look the same as the cover image even though it has another image inside of it. You can also extract the watermarked image from the cover image.
